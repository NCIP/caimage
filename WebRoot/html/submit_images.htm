What's New?
June 2010
 Starting with this release, the caIMAGE application utilizes aDORe djatoka, a Java-based open source image server with an attractive feature set. djatoka provides compression and region extraction of JPEG 2000 images, URI-addressability of regions, and support for a rich set of input/output image formats (e.g. BMP, GIF, JPG, PNG, PNM, TIF, JPEG 2000). djakota replaces the previously used Lizardtech server. Images stored in the Lizardtech specific format .sid have been lossless converted to work with the new image server.
