Additional Features
We would like to introduce a few features that we have build into the application to make the submission process more convenient.

Organ / Tissue Selection
Select the organ system in the first drop-down list. The content of the second drop-down list will change depending on your selection.
This selection mechanism can be found on the
 Cell lines page
 Histopathology pages
