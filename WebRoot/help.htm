Help - How to view MrSid images

PC and Internet Explorer and MrSid

If you click on the thumbnail of the image the plug-in installation will start automatically. After a few moments the image should be available for viewing in the MrSid image viewer.

In case the automatic installation did not work, follow these guidelines (courtesy of Lizartech support team)
Installation Overview for Windows:
First, verify that your computer meets the following system requirements:

Windows
Microsoft® Windows 95/98/2000 or Windows NT® Service pack 6
200MHz PC processor
Minimum of 64MB RAM
Internet Explorer 4.0 or greater, or Netscape® Navigator 4.5-4.7

Note that AOL and Netscape 6 browsers do not support MrSID at this time.
Second, locate the browser plugin on either the LizardTech Web site at http://www.lizardtech.com/browser. Save the file out to your computer and run the installation executable from there.
The MrSIDBrowserPlugin.exe should install the NPMrSID.dll file to your browsers /plugins directory. If it seems that the plugin has not been completely installed, reboot your machine and verify that the NPMrSID.dll is in this location. NPMrSID.dll can be manually copied to the /plugins folder if necessary.

PC and Netscape 4.7 and MrSid

You need to download the plug-in from the lizardtech website at http://www.lizardtech.com/browser
After the file has been saved on your computer run the installtion by double-clicking on the file and follow the instructions provided by the installation wizard.

PC and Netscape 6 and MrSid

The currently available version of the MrSid plug-in does not support Netscape 6. We learned from the lizardtech support team that a new version of the plug-in will be soon available. This version will support Netscape 6.

Mac and MrSid

Installing MrSID Browser Plug-in on Mac:
Verify that your computer meets the following system requirements:

Macintosh
Apple® G3 or later
Mac® OS 9 or later
20 MB allocated to Web browser
5 MB hard disk space
Internet Explorer 5.0 or greater or Netscape® Navigator 4.5-4.7

Download the Macintosh MrSID Browser Plugin from our Web site at http://www.lizardtech.com/browser .
Run the MrSID plugin installer.
The installer should place a MrSID plugin file in your browsers /plugins folder. If this plugin does not exist in the /plugins folder for the browser you are using then MrSID has not been installed. The MrSID plugin file can be manually copied to the /plugins folder if necessary.
Always reboot machine after installing.
