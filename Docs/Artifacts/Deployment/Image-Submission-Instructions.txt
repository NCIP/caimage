
Instructions for manually submitting images to caIMAGE.

1) Ensure the image file to be submitted is accessible from the image server on which the submission application will be run.
2) In the caIMAGE installation directory (corresponding to the trunk of the caIMAGE SVN repository https://ncisvn.nci.nih.gov/svn/caimage/trunk/), 
   navigate to the /utilities directory.
3) Execute the following command:
   java -jar caimage-submit.jar <filename>
4) When prompted, enter the species and description information pertaining to the image.
5) Note the location to which the image file should be copied, and copy the file to that location.
